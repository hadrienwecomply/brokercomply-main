# Annexe 2 · Emplacements tolérés des mentions, par format

Indique, pour chaque mention obligatoire, OÙ elle peut légalement figurer selon le support. Détermine le verdict quand une mention est absente du visuel analysé :

- Emplacement requis = **visuel** et mention absente du visuel → `non_conforme`
- Emplacement toléré = **accompagnement / profil / landing** et cet élément n'a pas été fourni pour analyse → `a_verifier` (indiquer où la mention doit se trouver)
- Emplacement toléré fourni et mention absente partout → `non_conforme`

Légende : **V** = visuel ou texte principal · **T** = texte d'accompagnement/légende · **P** = profil de la page · **L** = landing page (usage normal) · **(L)** = landing page en repli si texte d'accompagnement techniquement impossible · **auto** = géré par la plateforme · **—** = non applicable

Formats audio/vidéo (TV, radio, vidéo) : les mentions doivent être lisibles ou clairement audibles (art. VII.124 §1 al. 3 CDE).
